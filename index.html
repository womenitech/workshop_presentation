What is HTML?

  • HTML describes the structure of Web pages
  • HTML elements are the building blocks of HTML pages
  • HTML elements are represented by tags
    Eg. < h1 > for "heading" , < p > for "paragraph", < div > tag defines a division or a section.
  • Browsers do not display the HTML tags, but use them to render the content of the page

Code declaration

The < head > element contains some important information about the document:


  • Meta tags
    (! important for SEO. Eg: description, keywords, etc.)
  • Google Analytics tracking code
  • Stores < style > and < link > elements.

  • CSS is a language that describes the style of an HTML document
  • CSS describes how HTML elements should be displayed
